# Crypto-Pulse-AI: Real-time Market Sentiment & Analysis 📈
**Crypto-Pulse-AI** is a high-performance Python utility designed for modern traders and developers. By combining real-time market data from **CoinGecko** with the reasoning power of **Google Gemini AI**, this tool provides instant sentiment analysis and technical summaries for any cryptocurrency.
## 🚀 Features
 * **Live Price Tracking:** Fetches high-precision data including 24h highs, lows, and percentage shifts.
 * **AI Sentiment Engine:** Leverages gemini-2.5-flash to interpret market volatility.
 * **Trend Forecasting:** Generates data-driven Buy/Sell/Hold suggestions.
 * **Minimalist CLI:** Clean, interactive terminal interface for rapid checks.
## 🛠️ Installation & Usage
 1. **Clone the Repo:**
   ```bash
   git clone [https://github.com/hammadvirk-dev/Crypto-Pulse-AI.git](https://github.com/hammadvirk-dev/Crypto-Pulse-AI.git)
   cd Crypto-Pulse-AI
   
   ```
 2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   
   ```
 3. **Run the Script:**
   ```bash
   python crypto_pulse.py
   
   ```
